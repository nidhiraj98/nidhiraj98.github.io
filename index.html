A Beginner's guide to CSS properties

This is a quick guide to the majorly used CSS properties and their values
Property Possible Values Description
Text
text-transform capitalize Transforms the first character of each word to uppercase
uppercase Transforms all characters to uppercase
lowercase Transforms all characters to lowercase
Click here for more
text-align left Aligns the text to the left
right Aligns the text to the right
center Aligns the text to the center
justify Stretches the lines so that each line has equal width
Click here for more
Font
font-family Arial, Georgia, ComicSansMS, etc. Changes the font of the text
font-size 10px, 2in, 1.2em, etc. Changes the size of the text
Click here for more on the units
font-weight bold, normal, 900 Changes the boldness of the font.
Value can vary between 100 - 900, where 400 is the same as normal and 700 is the same as bold
opacity 0, 0.1, 1 Changes the opacity of the text.
It can take any value between 0 and 1, 0 being invisible and 1 being opaque
